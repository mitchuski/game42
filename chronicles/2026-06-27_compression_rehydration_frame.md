# Chronicle — the game as compression ⇄ rehydration (knowledge → promise → trust)

**Date:** 2026-06-27
**Status:** framing (Mitch's articulation) + first build step.

## The frame
The Game of 42 is the **shared-understanding key** to the knowledge base — the join between the
**myth layer** and the **math layer** we generate. It is one stage in a chain of *compressions
and rehydrations* of the same substance:

```
spellweb artefacts  →  a City Key  →  a star you control  →  a game of 42  →  a trust graph
   (myth)              (compression)    (identity)            (governance)     (network)
```

Each arrow is a **compression** (many → a κ-addressed seal) that can be **rehydrated** (the seal
→ back into a shape you can read, edit, and walk). The Three Graphs run underneath:
**knowledge → promise → trust** — what you *could* commit, what you *did* commit (the 42 VRC
edges), what was *kept* (the sealed shape). The game makes that flow editable and carryable.

## What this means for the build
- A **node is a slot for a compression.** In `my42`, each station can now hold: a chosen
  persona (6 axes → 42), a role in your words, free-text **information**, and an **imported City
  Key / JSON** (its name + κ docked onto the node). The node is where a compression rehydrates
  into the game, and where the game re-compresses (its own seal).
- **Same κ everywhere.** The κ that addresses a City Key, a game seal, a node's attached JSON is
  the *same canonical SHA-256* (`hash.js`) — so a thing compressed in one form (artefact, key,
  star, game) carries its identity into the next. That is what makes the chain interoperable
  rather than four separate apps (see `2026-06-27_interoperability_plan.md`).
- **Myth ⇄ math.** The personas, the proverbs, the star = myth; the lattice, the κ, the fold,
  the seal = math. The game is the membrane: edit in your own language (myth), and it
  compresses to a verifiable address (math), and back.

## Implication / next
- A node's attached key/json should be able to **rehydrate into its own sub-game** (the fractal
  seam): dock a City Key → it becomes a child game of 42 at that node. (Today: name+κ docked;
  child render deferred — same posture as the spec's Decision 3.)
- The **/grid** is where these compressed games meet as a network; the **constellation** is where
  many rehydrate into one greater star. Keep all three speaking the one κ.
